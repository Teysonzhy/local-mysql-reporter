# 财务报表助手 — 客户使用教程

---

## 一、系统概述

本系统是一款面向培训机构/门店的**财务报表管理工具**，核心功能包括：

- **会员管理** — 会员信息录入、查询、导入导出
- **教练管理** — 教练信息、课时安排、约课记录、课时费结算
- **财务月报** — 每月收支台账、纯利统计、公司账户、月报生成与导出
- **常规报表** — 授课日记、日常开支、体验课、请假记录、会员课时统计
- **报表分析** — 可视化图表概览、透视分析
- **数据导入** — 批量导入 Excel/CSV 数据

---

## 二、快速开始

### 2.1 数据库连接

首次使用需配置 MySQL 数据库连接：

1. 点击左侧导航栏 **「数据连接」**
2. 确认连接状态显示 **「已连接」**（绿色）
3. 查看 **「业务表行数」** 确认各表已有数据

> 如果连接失败，请检查 `.env` 文件中的数据库配置（地址、端口、用户名、密码、数据库名）。

### 2.2 数据导入（首次使用）

如果已有历史数据（Excel/CSV 文件）：

1. 点击左侧导航栏 **「导入数据」**
2. **方式一：一键批量导入**
   - 点击 **「选择目录」**，选中包含所有模板文件的文件夹
   - 系统自动识别文件名中的表名并导入
3. **方式二：单表导入**
   - 选择目标表 → 选择文件 → 点击 **「导入」**

## 三、会员管理

> 导航路径：左侧 **「会员管理」**

### 3.1 查询会员

- 在搜索栏输入条件进行筛选：
  - **会员卡名称** — 模糊搜索
  - **电话** — 模糊搜索
  - **教练** — 模糊搜索
  - **剩余课时** — 设置最小值/最大值范围
- 点击 **「查询」** 刷新列表

### 3.2 新增会员

1. 点击 **「新增」** 按钮
2. 填写会员信息：
  - **会员卡名称**（必填）
  - 电话、名称备注、教练、备注、剩余课时、流失说明（选填）
3. 点击 **「保存」**

### 3.3 修改/查看/删除

- **修改**：点击列表中的 **「编辑」** 图标，修改后保存
- **查看**：点击 **「查看」** 图标，只读模式浏览
- **删除**：点击 **「删除」** 图标，确认后删除

### 3.4 导入会员数据

1. 点击 **「导入」** 按钮
2. 选择 Excel/CSV 文件（需包含中文表头，如"会员卡名称"、"电话"等）
3. 系统自动匹配表头并批量导入

### 3.5 导出会员列表

点击 **「导出」** 按钮，选择保存路径，导出为 `.xlsx` 文件。

## 四、教练管理

> 导航路径：左侧 **「教练管理」**，包含 3 个标签页

### 4.1 教练信息

管理教练基本信息（姓名、电话、职位、门店、课时费单价、分成比例、状态）。

- **新增教练**：点击 **「新增」**，填写信息后保存
- **修改教练**：点击 **「编辑」** 图标
- **删除教练**：点击 **「删除」** 图标
- **导出列表**：点击 **「导出」** 按钮

### 4.2 课时安排

管理课程和约课记录。

#### 课程管理
1. 选择月份，点击 **「查询」** 加载当月课程
2. 点击 **「新增课程」**，填写：
   - 教练、课程名、上课时间、时长、最大人数、单课价值、状态
3. 点击约课人数可查看该课程的约课详情

#### 约课记录
1. 在课程详情弹窗中管理约课
2. 点击 **「新增约课」**，填写约课人、会员卡、约课时间、状态
3. 约课状态：**已约 → 已签到 → 已取消**

#### 课时费计算
- 点击 **「计算课时费」** 按钮
- 计算公式：**单课价值 × 约课人数 × 分成比例%**

### 4.3 课时费结算

1. 选择月份，查看当月课时费报表
2. 报表显示：教练、课程、上课时间、约课人数、课时费、计算公式、状态
3. **批量结算**：点击 **「批量结算」**，将当月所有"待结算"记录标记为"已结算"
4. 支持导出课时费报表

## 五、财务月报（核心功能）

> 导航路径：左侧 **「报表」→「财务月报」**

### 5.1 按月查看

1. 在页面顶部选择 **年月**（如 2026-04）
2. 系统自动加载该月的月报数据

### 5.2 支出明细

- 表格按 **分类** 自动分组，相同分类的行合并显示
- 每个分类末尾自动生成 **小计行**（浅蓝色背景）
- 表格底部显示 **合计**
- **添加行**：点击表格下方的 **「+」** 按钮
- **删除行**：点击行末的 **「×」** 按钮
- **自动保存**：修改后 800ms 自动保存，无需手动保存

### 5.3 收入明细

操作方式与支出明细相同，按分类分组并自动生成小计。

### 5.4 纯利统计

填写以下字段：
- **纯利** — 当月净利润
- **总耗课价格** — 正式课消耗总金额
- **体验课价格** — 体验课收入
- **教练工资** — 教练薪酬支出
- **现金流支出** — 其他现金流出
- **退课手续费** — 退课产生的手续费

### 5.5 公司账户

- **公司账户名** — 账户名称
- **上月余额** — 期初余额
- **售课金额** — 当月售课收入
- **其他收入** — 非售课收入
- **支出** — 当月总支出
- **转入银行卡** — 转出金额
- **期末余额** 自动计算：上月余额 + 售课金额 + 其他收入 - 支出 - 转入银行卡

### 5.6 继承上月

点击 **「继承上月」** 按钮，将上月的全部数据（主表 + 支出明细 + 收入明细）复制到当月，方便快速录入。

### 5.7 生成月报

点击 **「生成月报」** 按钮：
1. 系统自动生成精美的 HTML 月报文件
2. 月报包含：KPI 卡片、收支明细表、分类汇总分析、盈利能力/人力成本/收支结构分析与建议
3. 文件保存到桌面，自动打开浏览器预览

### 5.8 导出 Excel

点击 **「导出」** 按钮：
1. 选择保存路径
2. 导出为 `.xlsx` 文件，包含 **支出明细** 和 **收入明细** 两个 Sheet
3. 导出格式：表头浅灰加粗居中、小计行浅蓝加粗、合并单元格保留、黑色边框

## 六、常规报表

> 导航路径：左侧 **「报表」→ 展开子菜单**

### 6.1 授课日记

记录每天的授课情况。

- **字段**：教练、课程、上课日期、上课时间、时长(分钟)、学员数、单课价值、备注
- **操作**：选择月份查询 → 新增/修改/删除 → 导出 Excel

### 6.2 日常开支

记录日常经营支出。

- **字段**：日期、支出类型、项目明细、支出金额(元)、备注
- **操作**：选择月份查询 → 新增/修改/删除 → 导出 Excel

### 6.3 体验课

管理体验课的预约和转化跟踪。

- **字段**：日期、体验课类型、体验课学员、来源、体验课价值、教练、状态、备注
- **状态流转**：待体验 → 已体验 → 已转化 / 已流失
- **操作**：选择月份查询 → 新增/修改/删除 → 导出 Excel

### 6.4 请假记录

记录教练请假情况。

- **字段**：教练、月份(YYYY-MM)、请假天数（支持 0.5 天）、备注
- **操作**：选择月份查询 → 新增/修改/删除 → 导出 Excel

### 6.5 会员课时统计

查看所有会员的课时数据。

- **字段**：会员卡名称、电话、教练、上月初始课时、本月新增课时、本月消耗课时、剩余课时
- **操作**：查询全部会员 → 导出 Excel

## 七、报表分析

### 7.1 报表概览

> 导航路径：左侧 **「报表」→「报表概览」**

提供 5 个可视化图表：

| 图表 | 说明 |
|------|------|
| **会员漏斗** | 累计会员 → 付费会员 → 有课时会员 的转化 |
| **付费课时月趋势** | 近 12 个月的约课人数和消耗课时变化 |
| **会员流失分析** | 流失原因分布（饼图） |
| **体验转化分析** | 已转化/已流失/待体验的比例（环形图） |
| **活动引流分析** | 各引流渠道的体验人数（柱状图） |

页面还包含：
- **KPI 卡片**：累计会员、付费会员、当前有课时、流失会员
- **激励等级表**：可配置 I/II/III 级的课时范围、工资、绩效比例
- **账务计算公式**：可自由编辑保存

### 7.2 报表透视分析

> 导航路径：左侧 **「报表」→「报表透视」**

高级数据分析工具：

1. **选择维度**：设置行维度和列维度（年/月/期间/科目编码/科目名称）
2. **透视表**：自动生成行列交叉汇总表，支持行合计和列合计
3. **钻取明细**：点击数值单元格，展开该交叉点的明细记录
4. **图表切换**：支持折线图/柱状图/饼图
5. **筛选**：按年份和关键词过滤

## 九、注意事项

1. **自动保存**：财务月报编辑后 800ms 自动保存，无需手动保存
2. **月份切换**：切换年月后数据自动加载，未保存的修改会丢失
3. **删除操作**：所有删除操作需二次确认，请谨慎操作
4. **数据导入**：导入前请确保文件格式正确（第一行为中文表头）
5. **导出格式**：所有导出均为 `.xlsx` 格式，需用 Excel 或 WPS 打开
6. **继承上月**：会覆盖当月已有数据，请确认后再操作
