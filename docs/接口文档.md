## 1. 教练管理模块（coach.rs）

### 1.1 list_coaches


**返回 `CoachRow`：**

| 字段 | 类型 | 说明 |
|------|------|------|
| id | number | 教练ID |

| name | string | 是 | 姓名 |

### 3.4 generate_ledger_data

从业务表加工数据写入月报（教练工资计算）。

```typescript
invoke<string>("generate_ledger_data", { targetMonth: string })
```

**计算逻辑：**
1. 从 `coach_stat` 获取教练总课时、总客价
2. 从 `incentive_grade` 匹配课时等级
3. 从 `coach_salary` + `salary_structure` 获取店长补贴、财务补贴
4. 计算：课价绩效 = total_fee × ratio/100，绩效工资 = grade.salary
5. 总工资 = 基础工资 + 课价绩效 + 绩效工资 + 店长补贴 + 财务补贴
6. 写入 `expense_item`（category="教练工资"），更新 `daily_ledger.coach_salary`

### 3.5 inherit_ledger

```typescript

```
