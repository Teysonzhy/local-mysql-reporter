# 健身房财务管理系统 - 技术方案

> 更新时间：2026-04-27

## 一、技术选型

| 层级 | 技术 | 说明 |
|------|------|------|
| 前端 | TypeScript + Vite + 原生 HTML/CSS | 轻量，无框架依赖 |
| 后端 | Rust + Tauri 2.x | 高性能桌面应用框架 |
| 数据库 | MySQL 5.7+ / 8.0 | 本地或远程连接 |
| 通信 | Tauri IPC（invoke） | 前后端通过命令调用通信 |

### Tauri vs Electron

| 对比项 | Tauri | Electron |
|--------|-------|----------|
| 安装包大小 | ~10MB | ~150MB |
| 内存占用 | ~50MB | ~200MB |
| 安全性 | 高（Rust 后端） | 中 |
| 跨平台 | Win/Mac/Linux | Win/Mac/Linux |

### 3.2 核心表关系

```
coach ←── coach_stat ──→ incentive_grade
  │              │
  │              └──→ daily_ledger ←── expense_item
  │                                     ←── income_item
  ├── coach_salary ←── salary_structure
  │
  └── course（教练的课程/约课记录）
         │
         └──→ coach_stat（统计计算来源）
```

### 3.3 关键业务逻辑

**教练工资计算公式：**
```
课时等级 = 根据 total_hours 匹配 incentive_grade.range
课价绩效 = total_fee × (grade.ratio / 100)
绩效工资 = grade.salary
总工资   = 基础工资 + 课价绩效 + 绩效工资 + 店长补贴 + 财务补贴
```

**数据来源：**
- 总课时 / 总客价 → `coach_stat`（从 course 表聚合）
- 课时等级 / 基础工资 / 绩效比例 → `incentive_grade`
- 店长补贴 / 财务补贴 → `coach_salary`（关联 `salary_structure`）

**教练统计 SQL（一次性关联计算）：**
```sql
INSERT INTO coach_stat (stat_month, coach_id, coach_name, total_hours, trial_hours,
  normal_hours, solo_hours, normal_fee, solo_fee, total_fee)
SELECT
  '2026-03',
  t1.id, t1.coach_name, t1.total_hours,
  COALESCE(t2.trial_hour, 0),
  t1.total_hours - COALESCE(t2.trial_hour, 0),
  t1.solo_hours + COALESCE(t2.trial_hour, 0),
  t1.total_fee - COALESCE(t2.trial_fee, 0),
  t1.solo_fee + COALESCE(t2.trial_fee, 0),
  t1.total_fee + t1.solo_fee
FROM (
  SELECT co.id, co.name AS coach_name,
    COUNT(DISTINCT c.id) AS total_hours,
    COUNT(DISTINCT IF(c.member_card LIKE '%暑假加油包%', c.id, NULL)) AS solo_hours,
    COALESCE(SUM(c.course_value), 0) AS normal_fee,
    SUM(DISTINCT IF(c.member_card LIKE '%暑假加油包%', 70, 0)) AS solo_fee,
    COALESCE(SUM(c.course_value), 0) AS total_fee
  FROM course c INNER JOIN coach co ON c.coach_id = co.id
  WHERE DATE_FORMAT(c.start_time, '%Y-%m') = '2026-03' AND c.status != 3
  GROUP BY co.id, co.name
) t1
LEFT JOIN (
  SELECT coach_name, COUNT(1) AS trial_hour, SUM(unit_value) AS trial_fee
  FROM trial_class WHERE DATE_FORMAT(class_date, '%Y-%m') = '2026-03'
  GROUP BY coach_name
) t2 ON t2.coach_name = t1.coach_name;
```

## 六、部署方式

### 本地数据库模式（推荐）

```
用户电脑
┌─────────────────┐    ┌─────────────────┐
│  Tauri 客户端    │───▶│  本地 MySQL      │
│  (前端 + Rust)   │    │  financial_reporting │
└─────────────────┘    └─────────────────┘
```

### 远程数据库模式

```
用户电脑                    云服务器
┌─────────────────┐        ┌─────────────────┐
│  Tauri 客户端    │──网络──▶│  远程 MySQL      │
└─────────────────┘        └─────────────────┘
```

### 环境配置（.env）

```env
DB_HOST=127.0.0.1
DB_PORT=3306
DB_USER=root
DB_PASS=your_password
DB_NAME=financial_reporting
```
